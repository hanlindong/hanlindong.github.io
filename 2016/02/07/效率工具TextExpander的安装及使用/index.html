效率工具 TextExpander 中文环境下的安装及使用

  • OSX
  • 效率

TextExpander 已算是 Mac 平台必装的一款效率类软件，它能将那些需要重复输入的内容（Content）保存，并给其预设一个缩写词（Abbreviation），当下次需要时，你只需输入设定好的缩写词就会自动展开缩写词，获得完整的文本内容。而 Markdown 这种轻量级的「标记语言」也越来越多的被写作爱好者、撰稿者广泛使用。

快过年啦，入手 TextExpander 一枚，但是发现在搜狗输入法（中文）下，不能触发 snippet ，即使是 shift 转为英文模式也不行，偏偏要换成系统英文输入法才生效。
Google 后发现，可以通过官方的应用TEIMPrefSetter设置为后可用，但是在版本 5 中提示：

The file “ Settings.textexpander ” couldn’t be opened because there is no such file.

再次 Google，找到一条解决方法：

TextExpander 5でスニペット実行後、直前の文字が消える問題を解決するTEIMPrefsetterがエラーを出してしまうのを解決する方法

尼玛日文，看不懂啊，借助 Chrome 内置万能的 Google 翻译，方法如下：

  1. TextExpander>首选项>同步>“TextExpander 5（Dropbox使用同步）。”
  2. 退出TextExpander
  3. 再次运行TEIMPrefsetter
  4. 将中文删除即可。

TextExpander 的使用可参见：
利用 TextExpander 提高在 Mac 上撰写 Markdown 的效率

编辑个人定制后，插入一条表格，输入 ;table 即可。
